# Roadmap

## v1.0 (Now - MVP)
- 8 compliance tools (GST, PAN, calculator, status, company, PIN, HSN, returns)
- MCP Streamable HTTP transport
- OAuth 2.0 flow
- SQLite persistence
- Free/Pro/Business plans
- Landing page

## v1.1 (Live Data)
- Integrate Appyflow for live GST verification
- Integrate Setu for live PAN verification
- Expand HSN database to full 5000+ codes
- Expand PIN database to all 15,000+ codes
- Add Razorpay for plan upgrades

## v1.2 (More Tools)
- GSTR-2A/2B reconciliation
- TDS rate lookup
- MSME/Udyam registration check
- E-way bill status
- Import/Export IEC verification

## v2.0 (Platform)
- Web dashboard: usage analytics, key rotation, team management
- Webhook support: get notified when a GSTIN status changes
- Bulk verification: CSV upload for batch GST/PAN checks
- API (REST) in addition to MCP, for non-AI integrations
- White-label: let agencies offer GSTMCP under their brand

## Future
- GSTIN to company intelligence: financial history, compliance score
- Document processing: extract GSTIN/PAN from invoice images
- Compliance calendar: remind when GST returns are due
