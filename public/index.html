Available Tools

Everything you need for Indian compliance

8 tools covering GST, PAN, company registry, and more.

GST

gst_verify

Verify any GSTIN and get trade name, legal name, registration status, business type, and address.

PAN

pan_verify

Validate a PAN number and return the holder's name, category (individual, company, trust), and Aadhaar linkage status.

Calc

gst_calculator

Calculate exact CGST, SGST, and IGST split for any amount and GST rate. Handles intrastate and interstate cases.

Status

gstin_status

Quick active/inactive/cancelled check for any GSTIN. Lighter and faster than full verification.

MCA

company_search

Search the MCA registry by company name or CIN. Returns registration date, status, capital, and directors.

PIN

pincode_lookup

Resolve any Indian PIN code to its city, district, state, and postal region. Instant, no API call needed.

HSN

hsn_lookup

Look up any HSN or SAC code to get the description and applicable GST rate. Search by code or keyword.

Returns

gst_return_status

Check if a GSTIN has filed GSTR1 and GSTR3B for recent months. See filing dates and compliance history.
